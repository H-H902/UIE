# UIE（水下图像增强）评估指标
## 一、非参考指标（无 GT 场景专属）
无需高质量参考图，直接分析增强后图像的自身特征，适配“真实水下场景缺乏标准 GT”的核心场景。

1. **UCIQE（Underwater Color Image Quality Evaluation）**
   - 核心作用：量化色彩均衡度、对比度与饱和度，针对水下蓝/绿偏色与低对比度问题。
   - 特点：专用于水下图像，对水体光学退化特征有很强针对性。

2. **UIQM（Underwater Image Quality Measure）**
   - 核心作用：综合评估图像清晰度、色彩自然度与整体对比度，贴近人类视觉对水下图像的质量判断。
   - 特点：无参考场景的基准指标，应用广泛。

3. **NIQE（Natural Image Quality Evaluator）**
   - 核心作用：基于自然图像统计特征判断增强图与“人类认知中自然场景”的契合度。
   - 特点：通用型指标，适用于跨场景对比，鲁棒性强。

4. **MUSIQ（Multi-scale Image Quality）**
   - 核心作用：从局部细节到全局结构进行多尺度感知质量评估，更接近人类的分层观察习惯。
   - 特点：兼顾细节和整体自然度，结果更贴近主观感受。

---

## 二、参考指标（有 GT 场景专属）
需要成对数据（退化水下图 + 高质量参考图），通过像素级对比来量化增强效果的精度。

1. **PSNR（峰值信噪比）**
   - 核心作用：衡量增强图与 GT 的像素级失真程度，值越高还原度越好。
   - 特点：对像素偏差敏感，量化精度高。

2. **SSIM（结构相似性）**
   - 核心作用：从亮度、对比度、结构三个维度衡量视觉结构一致性。
   - 特点：比 PSNR 更贴近人类视觉感受，避免“像素误差小但视觉效果差”的评估偏差。

3. **MAE（平均绝对误差）**
   - 核心作用：计算像素绝对差值的均值，反映整体像素偏差水平。
   - 特点：计算简单，对极端偏差的敏感度低于 MSE，结果更稳健。

4. **MSE（均方误差）**
   - 核心作用：像素差值的平方均值，对严重失真区域更敏感。
   - 特点：是计算 PSNR 的基础，数值越低代表效果越优。

---

## 三、感知分数（PS↑，主观感知类指标）
跨越“有无 GT”的灵活指标，聚焦“人类主观视觉体验”，弥补纯像素级指标的局限。

- 核心定义
  - 有 GT 场景：评估增强图与 GT 在色彩、纹理、场景氛围上的主观一致性。
  - 无 GT 场景：判断增强图本身的视觉自然度（如是否符合真实水下光影规律）。
- 特点：与人类主观评分相关性最高，是算法落地实用性的关键评估依据。
