UIE（水下图像增强）评估指标体系
量化水下图像增强算法效果的核心指标集合，按 “是否依赖高质量参考图（GT）” 分为三大类，适配不同评估场景。
一、非参考指标（无 GT 场景专属）
无需高质量参考图，直接分析增强后图像的自身特征，适配 “真实水下场景缺乏标准 GT” 的核心痛点。
1. UCIQE（水下彩色图像质量评价）
核心作用：针对水下图像蓝 / 绿偏色、低对比度等问题，量化色彩均衡度、对比度及饱和度。
特点：水下场景专用，对水体光学退化特征针对性极强。
2. UIQM（水下图像质量度量）
核心作用：综合评估增强图的清晰度、色彩自然度与整体对比度，贴近人类视觉对水下图像的质量判断。
特点：水下增强领域无参考场景的 “基准指标”，应用范围最广。
3. NIQE（自然图像质量评价）
核心作用：基于自然图像的统计特征库，判断增强图与 “人类认知中自然场景” 的契合度。
特点：通用型指标，不局限于水下场景，鲁棒性强，可跨场景对比。
4. MUSIQ（多尺度图像质量评价）
核心作用：从 “局部细节” 到 “全局结构” 多尺度评估感知质量，还原人类观察图像的分层习惯。
特点：兼顾细节清晰度与整体自然度，结果更贴近主观感受。
二、参考指标（有 GT 场景专属）
需配套 “退化水下图 + 高质量参考图（GT）” 的成对数据，通过像素级对比量化增强效果的精准度。
1. PSNR（峰值信噪比）
核心作用：计算增强图与 GT 的像素级失真程度，数值越高代表失真越小、还原度越高。
特点：图像复原领域经典指标，对像素偏差敏感，量化精度高。
2. SSIM（结构相似性）
核心作用：从 “亮度、对比度、结构” 三个维度，衡量增强图与 GT 的视觉结构一致性。
特点：相比 PSNR 更贴近人类视觉，避免 “像素误差小但视觉效果差” 的评估偏差。
3. MAE（平均绝对误差）
核心作用：计算增强图与 GT 的像素绝对差值均值，反映整体像素偏差的平均水平。
特点：计算简单，对极端像素偏差的敏感度低于 MSE，结果更稳健。
4. MSE（均方误差）
核心作用：计算增强图与 GT 的像素差值平方均值，对严重失真区域的大偏差更敏感。
特点：PSNR 的计算基础，数值越低代表增强效果越优。
三、感知分数（PS↑，主观感知类指标）
跨越 “有无 GT” 的灵活指标，聚焦 “人类主观视觉体验”，弥补纯像素级指标的局限性。
核心定义
有 GT 场景：评估增强图与 GT 在色彩、纹理、场景氛围上的 “主观感知一致性”；
无 GT 场景：直接判断增强图本身的 “视觉自然度”（如是否符合真实水下光影规律）。
特点
结果与人类主观评分相关性最高，是 “算法落地实用性” 的核心评估依据。
四、指标适用场景速查表
评估场景	核心推荐指标	辅助指标
真实水下无参考增强	UCIQE、UIQM	NIQE、MUSIQ
有标准 GT 的算法研发	PSNR、SSIM	MAE、MSE
算法落地前的主观验证	PS（感知分数）	搭配对应场景的核心指标
